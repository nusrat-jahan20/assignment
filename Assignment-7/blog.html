1. What is the difference between map(), filter(), forEach(), find()?

Answer: map(): Returns a new array by applying a function to each element in the original array.
filter(): Returns a new array with elements that meet a specified condition.
forEach(): Executes a provided function once for each array element.
find(): Returns the value of the first element in an array that satisfies a provided condition.

2. What is the difference between localStorage, sessionStorage, cookies ?

Answer: localStorage: stores data in the browser indefinitely until manually cleared.
sessionStorage: stores data in the browser until the window or tab is closed.
cookies: stores small text files on the user's computer, sent with each HTTP request.

3. What is event loop? How it works?

Answer: The event loop in JavaScript is a way of handling multiple tasks at the same time, so that long-running tasks don't block the main thread and slow down the application. It manages a queue of tasks to be executed one after the other.

5. How chooseQuiz() function works?

Answer: The function chooseQuiz() takes two parameters: 'index' and 'givenAns'. The function checks if the current question already exists in the 'answers' array. If the current question already exists in the 'answers' array, it updates the existing answer with the new answer. If the current question does not exist in the 'answers' array, it adds the new answer to the array. Finally, the function calls the 'displayAnswers' function to display the updated answers array at right side of the page.
